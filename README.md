# ai-study-timer
It is a smart study timer  your  choice.
This project is a simple Python-based study planning tool that helps users organise their study time efficiently.

## Features
- Takes total study time as input
- Allows multiple subjects
- Automatically distributes study time
- Includes optional Pomodoro cycle calculation
- Provides structured study and break plan

## How It Works
The program uses basic logic to divide study time across subjects and suggests a balanced schedule. It also supports Pomodoro-style study cycles for better productivity.

## Technologies Used
- Python
